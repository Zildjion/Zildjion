About Us

WebDev Solutions is a collection of webpages made in order to satisfy one thing: HTML-CSS Education.

Cascading Style Sheets (CSS) is a core web technology that is used to manage how HTML documents look visually. It provides guidelines for a web page's layout, colors, fonts, and other design aspects, enabling web designers to construct websites that are both aesthetically pleasing and user-friendly. Modern web design relies heavily on CSS, which is necessary for producing user interfaces that are both appealing and responsive.

This page was made with the sole objective to help others explore and understand various methods of applying CSS (Cascading Style Sheets) to HTML web pages. This serves as your avenue to conduct online research to discover and learn the different techniques for styling web content using CSS.

This webpage applies at least one unique style to each HTML element on the webpage. It makes sure that every element (e.g., paragraphs, headings like 'h1', 'h2', etc.) has a distinct style. For instance, if there are multiple 'p' (paragraph) tags, the same style should apply consistently to all the 'p's.

This is done to adhere to original design specifications and instructions, however I took the liberty to apply other styles for the pages linked to this 'home' page.

The University of the Philippines Mindanao (also referred to as UPMin or UP Mindanao) is a public research university, serving as the sixth constituent unit of the University of the Philippines System.

UP Mindanao is located in Barangay Mintal, in a setting that is unique in the booming Davao City. It is surrounded by a landscape that is free of intrusive urban and commercial distractions.

UP Mindanao is the only constituent university of the UP System that was created through legislative action. Republic Act 7889 formally created UP Mindanao on February 20, 1995.

The university was later formally recognized as an independent constituent university by the Board of Regents of the UP System on March 23, 1995.

Its main focus of education is Mindanao studies through an affirmative action program in the Autonomous Region in Muslim Mindanao to attract Muslims and Lumad students, aside from the marginalized and deserving students.

The university was first named University of the Philippines in Mindanao to declare the arrival of the state university in Mindanao after a long period of waiting by eager alumni and students.

This was later replaced to its present name due to the request of the UP System for formalities' sake.
